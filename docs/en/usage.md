# Usage guide

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start a local Hardhat node:
   ```bash
   npx hardhat node &
   ```
3. Deploy contracts locally:
   ```bash
   npm run deploy:local
   ```
4. Run tests if needed:
   ```bash
   npm run test
   ```

Before deploying to public networks create a `.env` file based on `env-example.js`.
